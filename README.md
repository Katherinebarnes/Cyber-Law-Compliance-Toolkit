# Cyber-Law-Compliance-Toolkit
A Python-based tool to generate Section 65B Certificates and validate digital evidence under the Indian IT Act.

# Cyber Law Compliance Toolkit (India)
Built by an **M.Tech (AI) & Cyber Law Scholar** to bridge the gap between engineering and legal compliance.

## Features
- **Digital Evidence Hashing:** Generates SHA-256 hashes to ensure data integrity.
- **Section 65B Draft Generator:** Automatically creates a draft certificate for Indian Courts.
- **Metadata Extraction:** Pulls technical details for forensic reports.

## Why this exists?
Under **Section 65B of the Indian Evidence Act**, digital evidence is only admissible if accompanied by a certificate. This tool automates the technical part of that process.
